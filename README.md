# Censoring and Blocking of Sensitive Content for Minors in Social Networks

## Data Availability Statement

The code used in this study is publicly available at:  
https://github.com/tanmayjayanthi2005-droid/censoring_and_blocking_of_sensitive_content_for_minors_in_social_networks

The dataset generated for this research cannot be publicly released due to ethical and legal restrictions involving sensitive content related to minors.
